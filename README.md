# Portfolio Optimization using Time Series Forecasting

## Project Overview
GMF Investments aims to optimize portfolio management strategies by integrating advanced time series forecasting models. This project utilizes historical financial data for Tesla (TSLA), Vanguard Total Bond Market ETF (BND), and S&P 500 ETF (SPY) to predict market trends and optimize asset allocation.

## Structure
- `data/`: Contains raw and processed financial data.
- `notebooks/`: Jupyter notebooks for EDA and modeling.
- `src/`: Source code for data loading, analysis, and modeling.
- `scripts/`: Utility scripts.
